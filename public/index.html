Contribute

You can extend Alexa Remote Desktop with your own plugins that can do exactly what you need

Building a plugin is a matter of creating a simple npm package, and tag it with the alexa-remote-desktop tag and upload it to the npmjs registery. Once it is there it can be found in the Alexa Remote Desktop application.

If you want to learn how to create a plugin visit the github page for more details
